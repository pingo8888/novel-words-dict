## ✨小说词库
开源、免费的桌面词库记录、分组、查询小工具，主要服务小说写作需要。

界面预览：
![](./docs/preview.png)


## ✨主要特性

1、阅读任意内容时，选中文本后，可用设定好的**全局快捷键**将选中文本添加入自定义词库

2、支持按词库、风格、名词类型、性别和关键字进行搜索，支持多个关键字，及关键字的模糊查找

3、支持Ctrl+左键点击词条，使用默认浏览器进行搜索

4、基于tauri框架开发，安装包、系统内存占用极低

5、随更新不断增加的内置词库

**注意：由于词条主要靠人工搜集与校验，无法保证100%准确，如有问题，欢迎提Issue。**


## ✨使用方法

1、下载最新版的release安装包进行安装，以后可随版本进行自动更新

2、在界面右上角的设置界面，设定自定义词库的目录，及界面取词快捷键

3、打开软件界面，在筛选区内选择，或在关键字输入框内输入关键字，进行筛选、查询
